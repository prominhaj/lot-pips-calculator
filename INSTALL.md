# Installation Guide

## üì± Desktop Installation (Chrome/Edge/Brave)

### Step 1: Download
1. Go to [Releases](../../releases)
2. Click on the latest release
3. Download `lot-pips-calculator.zip`

### Step 2: Extract
- Right-click the ZIP file
- Select "Extract All" or "Extract Here"
- Remember the folder location

### Step 3: Install
1. Open your browser
2. Type in address bar:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
3. Turn ON "Developer mode" (top-right corner)
4. Click "Load unpacked" button
5. Select the extracted folder
6. Done! ‚úÖ

## üì± Android Installation (Kiwi Browser)

### Step 1: Install Kiwi Browser
- Open Google Play Store
- Search "Kiwi Browser"
- Install it

### Step 2: Download Extension
1. Open Kiwi Browser
2. Go to [Releases](../../releases)
3. Download `lot-pips-calculator.zip`
4. Extract using Files app or any file manager

### Step 3: Install Extension
1. In Kiwi Browser, type: `chrome://extensions/`
2. Enable "Developer mode"
3. Tap "Load unpacked"
4. Navigate to extracted folder
5. Select it
6. Done! ‚úÖ

## üçé iOS (Not Supported)

## ‚úÖ How to Use

1. Open [TradingView](https://tradingview.com)
2. Open any chart
3. Add "Long Position" or "Short Position" tool
4. Click on the position to open settings
5. Scroll down - you'll see "Calculate" button
6. Fill in your risk amount
7. Click "Calculate" to see:
   - Lot Size
   - Margin Required
   - Stop Loss (pips + $)
   - Take Profit (pips + $)
   - Risk/Reward Ratio

## ‚ùì Troubleshooting

**Extension not showing?**
- Make sure you're on TradingView website
- Refresh the page
- Check if extension is enabled in `chrome://extensions/`

**Calculate button not appearing?**
- Make sure you added Long/Short Position tool
- Open the position settings
- Scroll to the bottom

**Wrong calculations?**
- Make sure all fields are filled (Entry, Stop, Risk)
- Check if symbol is detected correctly
